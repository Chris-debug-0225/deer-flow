## 🛠️ 使用说明

### 查看 Mermaid 图表

所有图表均使用 Mermaid 语法编写，可以通过以下方式查看：

1. **GitHub/GitLab**：自动渲染 Mermaid 图表
2. **VS Code**：安装 Mermaid 插件预览
3. **Markdown 编辑器**：大多数现代编辑器支持 Mermaid
4. **在线渲染器**：使用 [Mermaid Live Editor](https://mermaid.live/)

### 编辑 Mermaid 图表

1. 使用文本编辑器打开 `.md` 文件
2. 找到 ` ```mermaid ` 标记的代码块
3. 编辑图表定义
4. 保存并预览

### 添加新图表

在现有文档中添加新图表：

1. 打开目标文档
2. 找到合适的位置
3. 添加新的 mermaid 代码块
4. 更新文档索引
