👋 这套文档是写给谁的

如果你是 Agent 开发小白，但有一定 Python 基础，想通过精读一个真实的、被广泛使用的开源框架来掌握 Agent 开发的设计思想与工程实现——那这里就是为你准备的。每一章都做到三件事：讲清概念、对照源码、给出可运行示例。

🗺️ 这个框架到底在解决什么问题

大语言模型（LLM）本身只会"输入文本 → 输出文本"。但我们想要的"智能体（Agent）"需要能调用工具、多步骤思考、与其他 Agent 协作、记住对话历史、在危险操作前暂停等待人类批准。DeerFlow 就是把这些能力用一套优雅的抽象封装起来，让你几行代码就能跑起一个能用的智能体。

用户输入 "帮我查天气并预订餐厅"
▼
Agent 循环 LLM 决策 → 调工具 → 看结果 → 再决策
▼
最终输出 结构化结果 / 自然语言回复

🧠 五个你必须先记住的名词

名词一句话理解类比
Agent带指令、工具、护栏的一个"智能员工"一名有岗位说明书的员工
SkillAgent 能调用的能力（查天气、查数据库…）员工手里的工具箱
Sub-Agent把任务转交给另一个更专业的 Agent把工单转给别的部门
Sandbox隔离执行环境，不合规就拦截门口的安检
Memory跨多轮对话保存历史记忆员工的工作记录本
