关键模块

DeerFlow 的核心模块和它们的职责。

后端核心模块

模块 位置 职责

Skills backend/app/skills/ Skill 系统

Gateway backend/app/gateway/ API 网关

  • backend/app/executor.py - 执行引擎
  • backend/app/memory.py - 内存管理
  • backend/requirements.txt - Python 依赖

高级路线：完整系统理解

从入口点开始追踪完整的请求流程：

  1. HTTP 请求进入 Gateway
  2. Gateway 路由到相应的 Agent
  3. Agent 分析任务
  4. Agent 选择和执行 Skills
  5. 结果存储到 Memory
  6. 返回响应给客户端
