在使用 DeerFlow 的过程中，可能会遇到各种问题。本章总结了常见问题、调试方法和解决方案，帮助你快速定位和解决问题。

常见问题

DeerFlow 使用中最常见的问题和解决方案。

问题 1：Agent 无法加载 Skill

症状： Agent 启动时报错，找不到 Skill。

原因： Skill 未正确注册或路径错误。

解决方案：

  1. 检查 Skill 文件是否存在
  2. 验证 Skill 注册代码
  3. 检查 Python 路径配置
  4. 查看日志获取详细错误信息

问题 2：LLM API 调用失败

症状： Agent 执行任务时 LLM 调用失败。

原因： API 密钥错误、网络问题或 API 配额不足。

解决方案：

  1. 验证 API 密钥是否正确
  2. 检查网络连接
  3. 检查 API 配额和使用情况
  4. 实现重试机制

问题 3：内存占用过高

症状： Agent 运行一段时间后内存占用不断增加。

原因： 内存泄漏或缓存未清理。

解决方案：

  1. 检查是否有循环引用
  2. 实现缓存清理机制
  3. 使用内存分析工具定位泄漏
  4. 定期重启 Agent

性能问题

常见的性能问题和优化方法。

性能问题对比

问题 症状 解决方案
响应慢 API 响应时间长 缓存、异步处理、模型优化
高 CPU 占用 CPU 使用率高 优化算法、并行化、减少计算
高内存占用 内存使用量大 清理缓存、流式处理、分批处理
高成本 API 调用费用高 缓存、模型选择、请求合并

优化建议

  • 缓存： 缓存常见问题的答案，避免重复计算
  • 异步处理： 使用异步 I/O 提高吞吐量
  • 批处理： 合并多个请求减少开销
  • 模型选择： 为不同任务选择合适的模型
  • 监控： 持续监控性能指标

错误解决方案

常见错误的快速解决方案。

ImportError: 找不到模块

# 问题：ImportError: No module named 'xxx' # 解决方案： # 1. 检查模块是否安装 pip install xxx # 2. 检查 Python 路径 import sys print(sys.path) # 3. 检查虚拟环境 source venv/bin/activate

AttributeError: 对象没有属性

# 问题：AttributeError: 'xxx' object has no attribute 'yyy' # 解决方案： # 1. 检查对象类型 print(type(obj)) # 2. 检查属性是否存在 print(dir(obj)) # 3. 检查初始化 print(obj.__dict__)
